# Contributing

## Local setup

```bash
pnpm install
pnpm build
pnpm test
```

## Rule contributions

Every new rule needs:

- Rule ID and documentation.
- Deterministic implementation.
- Evidence.
- Severity and confidence.
- Remediation and validation steps.
- Positive and negative fixture coverage.
- Report rendering coverage.

## Report contract

Reports must be generated by the renderer and must pass:

```bash
seo-polish report lint ./seo-polish-report --strict
```
