# @seo-polish/adapters

Framework-aware, bounded change-set planning and application. Adapters are report-driven, path-confined,
proposal-first, and require explicit approval for crawler policy, indexing, canonical, auth, commerce, business
data, AI policy and public agent capability changes.

The package detects Next.js, Astro, Nuxt, SvelteKit, Remix, Docusaurus and static sites. Every proposed write carries
content hashes, remains inside the selected repository, refuses symlink escapes and stale updates, and is written to
`change-set.json` plus `change-set.diff` before application.
