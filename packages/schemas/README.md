# @seo-polish/schemas

Shared TypeScript contracts, report section definitions and lightweight schema validators.

Includes `ReportDashboard`, the stable execution cockpit model used by `report-dashboard.json`,
`AgentReview` and `AgentReviewInput`, the mandatory evidence-linked agent review contracts used by
`agent-review.json` and `agent-review-input.json`,
`BrowserEvidenceReport`, the bounded browser-rendering evidence model used by `browser-evidence.json`,
and `FieldDataReport`, the CrUX/Search Console/RUM evidence model used by `field-data.json`, the
static HTML report and the final agent execution plan.
